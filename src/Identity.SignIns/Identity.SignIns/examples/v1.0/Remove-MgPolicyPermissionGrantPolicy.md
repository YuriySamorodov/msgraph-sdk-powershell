### Example 1: Using the Remove-MgPolicyPermissionGrantPolicy Cmdlet

```powershell
Import-Module Microsoft.Graph.Identity.SignIns
Remove-MgPolicyPermissionGrantPolicy -PermissionGrantPolicyId $permissionGrantPolicyId
```
